# ft_printf
> 42 school project developed by Ernest Balañá

## Description
`ft_printf` is a custom recreation of the standard C `printf` function. The main goal of this project is to learn how to handle variadic arguments in C and build a flexible formatting utility from scratch.

## Features
- **Variadic Arguments:** Efficient processing of an unexpected number of parameters using `stdarg.h`.
- **Supported Conversions:** Handles standard flags: `%c`, `%s`, `%p`, `%d`, `%i`, `%u`, `%x`, `%X`, and `%%`.
- **Memory Efficient:** Direct string parsing and buffer management without memory leaks.
